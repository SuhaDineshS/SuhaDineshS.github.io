Shadow Path Prototype

A puzzle mechanic centered around manipulating sunlight direction and rotating the world to create walkable shadow paths.

Project Overview

Genre: Puzzle
Platform: PC (Unity)

A mechanic-first prototype exploring new ways to solve puzzles by shaping shadows rather than moving the player character.

Key Roles (Solo Project)
• Designed sunlight manipulation and stage rotation to control shadows.
• Built 3 gameplay levels introducing new mechanics
• Implemented pathfinding and auto-walk systems
• Developed raycast-based shadow pathfinding

Tools Used: Unity · C# · Navmesh

Core Mechanics

• Rotate the entire stage using A / D
• Change time of day using W / S
• Shadows dynamically form walkable paths
• Press Space → Auto-walk to goal when path exists
• Expandable towers that modify geometry in Level 3

Challenges Faced

• Accurate ray-based shadow path detection
• Rotation of the stage caused calculation erros and navmesh inconsistencies.

Key Takeaways

• Learned lighting and shadows.
• Developed mechanic-first puzzle design
• Experience with Navmesha and pathfinding.
