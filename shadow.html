Shadow Path Prototype

A puzzle system built around manipulating time of day and sun direction to create walkable paths made from shadows. This includes three showcase levels demonstrating rotation mechanics, shadow path detection, and an expanding tower system.

1. Level 1 — Time of Day Control & Stage Rotation

The first level introduces the core ideas:

• Rotate the entire stage using A / D keys.
• Change the time of day using W / S keys, which adjusts the sun angle in real time.
• Shadows stretch, shrink, and reposition dynamically, creating new traversal paths.

This level proves the core mechanic: the player doesn’t move the character — they move the world and the light to make a path and press Space which moves the character to the goal automatically if a path exists.

2. Level 2 — Harder Shadow Navigation

The second level expands on the core mechanic by designing a more indirect path. The player must adjust rotation and time of day in combination to make the required path.

3. Level 3 — Expanding Tower Mechanic

This level introduces a new mechanic: expandable towers that rise to complete shadow paths.

The player can position the shadows precisely, but sometimes the world geometry must be modified. These towers act as adjustable platforms that complete otherwise impossible shadow routes.
