Ragdoll Blending System

A physics-driven gameplay prototype focused on limb-based force application, ragdoll takeover, and animation recovery.

Project Overview

Genre: Gameplay System Prototype
Platform: PC (Unity)

A reactive character system where animation and physics blend seamlessly based on player interaction and real-time forces.

Key Roles (Solo Project)
• Implemented limb-specific raycast detection
• Built full ragdoll transition system
• Designed recovery logic based on fall direction
• Tuned forces for realistic physical feedback

Tools Used: Unity · Blender · C#

Core Mechanics

• Limb-specific impacts based on raycast hits
• Dynamic force application to individual rigidbodies
• Full-body collapse with animation-to-physics blending
• Fall direction detection (face-up vs face-down)
• Smooth re-entry into locomotion animations

Challenges Faced

• Achieving realistic physical feedback without instability
• Creating smooth ragdoll-to-animation transitions
• Maintaining consistent movement direction after standing up

Key Takeaways

• Strong understanding of Unity’s humanoid rigs
• Experience building hybrid animation–physics systems
• Learned the importance of precise blending and timing
